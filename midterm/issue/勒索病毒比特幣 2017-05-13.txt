比特幣勒索病毒凶猛，微軟對已停止支持的 Windows XP 和 Windows 2003 發特別補丁5月13日，NSA 「永恆之藍」勒索蠕蟲肆虐終於引起了大家的重視，雷鋒網編輯連發多篇稿件後，一覺（午覺）醒來發現這個勒索蠕蟲病毒連名字都變了，成了「比特幣勒索病毒」。然後，就發現它的風刮到了銀行ATM 機、德國火車站……，很多人表示，受到了驚嚇。本來，Win7及以上版本的系統要確認是否安裝MS07-010補丁，如沒有安裝則受威脅影響。Win7以下的Windows XP/2003目前沒有補丁，只要開啓SMB服務就受影響。結果因為這波影響實在太大，微軟被炸出來了，剛才決定對已經停止支持的 Windows XP 和 Windows 2003 發佈特別補丁。
